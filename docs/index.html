Installation

Quick setup with uv and pip for modern Python development. 

# Install with pip
pip install mcp-manager

# Or install from source
git clone https://github.com/kairin/mcp-manager.git
cd mcp-manager
uv venv && source .venv/bin/activate
uv pip install -e .
🚀

Quick Start

Initialize and audit your MCP configuration in minutes. 

# Audit current setup
mcp-manager audit

# Initialize global config
mcp-manager init --global

# Check server health
mcp-manager status
💻

CLI Commands

Comprehensive command-line interface for all operations. 

# Add new server
mcp-manager add context7 \
  --type http \
  --url https://mcp.context7.com/mcp

# Update all servers
mcp-manager update --all
